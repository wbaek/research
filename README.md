# research

## pre-requirements
### for Mac

* install ImageMagick 7.0.7-14

```bash
brew install dependencies/imagemagick.rb
pip install -r requirements_pre_for_mac.txt
```

### for ubuntu

* install ImageMagick
```bash
apt-get install -y libsm6 libmagickwand-dev
```

## install
```
pip install -r requirements.txt
```
